RSA (Rivest–Shamir–Adleman) is an asymmetric cipher — a public key encrypts, a private key decrypts. Anyone can encrypt a message to you; only you can read it. Used in TLS, SSH, and email (PGP). We use RSA-2048 with OAEP padding.

SHA-256 (Secure Hash Algorithm) is a one-way hash function. It converts any input into a fixed 256-bit fingerprint. It cannot be reversed. Used to store passwords, verify file integrity, and power blockchains. Salting adds a random value before hashing so identical inputs produce different hashes.
